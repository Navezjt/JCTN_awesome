# 直播源相关资源汇总

现在各个宽带运营商都有 IPTV 业务，不用安装有线电视就可以高清无卡顿的看电视。

没有开通 iptv 可以使用抓取的直播源地址，复制到电脑、手机、电视网络盒子等设备上看，如果动手能力强也可以自己抓包获取直播源。

由于版权原因很多电视直播软件播放时会卡顿、画面不清晰。
